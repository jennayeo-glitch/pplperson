# ppl person - 파티플래너 랜딩페이지

파티플래너 비즈니스를 위한 랜딩페이지입니다.

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)
- Vite

## 주요 기능

- 이벤트 카드 표시 및 필터링
- 카카오 소셜 로그인
- 이벤트 상세 페이지
- 참여 기능
- 반응형 디자인

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## GitHub Pages 배포

1. GitHub 저장소 생성
2. 코드 푸시
3. Settings > Pages에서 배포 설정
4. Source를 "GitHub Actions" 또는 "main branch /docs" 선택
